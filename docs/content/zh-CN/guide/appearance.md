# 显示与渲染

显示设置分为场景变换、播放行为和渲染效果。建议先在全局配置设置共同默认值，再只为特殊模型添加覆盖。

## 场景变换

| 选项 | 作用 |
| --- | --- |
| 显示 | 决定模型是否出现在目标场景 |
| 锚点 | 选择模型相对屏幕边界或中心的基准点 |
| 水平/垂直偏移 | 相对 1920×1080 参考画布保存的位置 |
| 缩放 | 等比调整模型大小 |
| 旋转 | 顺时针旋转角度 |
| 透明度 | `0` 为完全透明，`1` 为完全不透明 |
| 显示层级 | 控制模型与其他 2D 内容的前后关系 |
| 鼠标交互 | 是否允许模型接收鼠标输入 |

运行时会根据窗口短边等比换算位置和大小，不会把模型拉伸到不同纵横比。

## 混合模式

- 正常：默认透明混合。
- 叠加：增强发光或高亮效果。
- 减色：从背景颜色中减去模型颜色。
- 正片叠底：让模型与背景相乘，整体通常更暗。
- 预乘透明度：用于已经按透明度预乘颜色的特殊素材。

如果不确定，应保持“正常”。

## 颜色滤镜

滤镜会先把整棵模型树合成，再统一处理，避免不同 Drawable 出现不一致颜色。

| 选项 | 常用范围 | 中性值 |
| --- | --- | --- |
| 色调 | RGBA | 白色 |
| 亮度 | `-1..1` | `0` |
| 对比度 | `0..4` | `1` |
| 饱和度 | `0..4` | `1` |
| 灰度 | `0..1` | `0` |
| 色相偏移 | 任意角度 | `0` |
| 反相 | `0..1` | `0` |
| Gamma | `0.01..10` | `1` |

一次只调整少量参数更容易定位效果。出现纯白、纯黑或颜色异常时，先恢复中性值，再检查模型纹理和 Shader。

## 画布裁切

裁切区域使用模型局部坐标，支持矩形、椭圆和圆角矩形。宽高必须大于 0；圆角矩形的边缘分段越高越平滑，
但也会产生更多几何顶点。裁切只影响整模型最终显示范围，不会修改 Cubism 自身的 Drawable 蒙版。

## 蒙版视口尺寸

该选项控制 Cubism Drawable 蒙版纹理尺寸，`0` 表示自动。提高数值可能改善高分辨率蒙版边缘，也会增加显存占用。
