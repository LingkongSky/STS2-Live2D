# 显示与渲染

显示设置分为场景变换、播放行为和渲染效果。建议先在全局配置设置共同默认值，再只为特殊模型添加覆盖。

## 场景变换

| 选项 | 作用 |
| --- | --- |
| 显示 | 决定模型是否出现在目标场景 |
| 锚点 | 选择模型相对屏幕边界或中心的基准点 |
| 水平/垂直偏移 | 相对 1920×1080 参考画布保存的位置 |
| 缩放 | 等比调整模型大小 |
| 旋转 | 顺时针旋转角度 |
| 透明度 | `0` 为完全透明，`1` 为完全不透明 |
| 显示层级 | 控制模型与其他 2D 内容的前后关系 |
| 鼠标交互 | 是否允许模型接收鼠标输入 |

运行时会根据窗口短边等比换算位置和大小，不会把模型拉伸到不同纵横比。

## 混合模式

- 正常：默认透明混合。
- 叠加：增强发光或高亮效果。
- 减色：从背景颜色中减去模型颜色。
- 正片叠底：让模型与背景相乘，整体通常更暗。
- 预乘透明度：用于已经按透明度预乘颜色的特殊素材。

如果不确定，应保持“正常”。

## 颜色滤镜

滤镜会先把整棵模型树合成，再统一处理，避免不同 Drawable 出现不一致颜色。
颜色使用可视化颜色选择器；亮度、对比度、饱和度、灰度、色相、反相与 Gamma 同时提供滑动条和数字输入，两者实时同步。

| 选项 | 常用范围 | 中性值 |
| --- | --- | --- |
| 色调 | RGBA | 白色 |
| 亮度 | `-1..1` | `0` |
| 对比度 | `0..4` | `1` |
| 饱和度 | `0..4` | `1` |
| 灰度 | `0..1` | `0` |
| 色相偏移 | 任意角度 | `0` |
| 反相 | `0..1` | `0` |
| Gamma | `0.01..10` | `1` |

一次只调整少量参数更容易定位效果。出现纯白、纯黑或颜色异常时，先恢复中性值，再检查模型纹理和 Shader。

## 实时预览

模型列表中的“预览并编辑”会打开独立画布。场景、参考分辨率、位置、缩放、旋转、混合模式、滤镜和画布蒙版都会立即反映到模型上。
默认模式下拖动画布移动模型，滚轮缩放模型，`Shift + 滚轮` 旋转模型。

## 画布蒙版

画布蒙版使用模型局部坐标，支持矩形、椭圆和圆角矩形，宽高必须大于 0。点击“启用蒙版并适配模型边界”可生成覆盖整个模型的初始区域。
开启“在画布上直接编辑蒙版”后，拖动可移动蒙版，滚轮可围绕中心缩放；滑动条与数字输入会同步更新。

圆角半径作用于圆角矩形。调整圆角半径时，编辑器会自动把类型切换为圆角矩形。椭圆与圆角边缘由着色器连续计算，不需要设置边缘分段。
该蒙版只裁切模型最终输出，不会修改 Cubism Drawable 自身的蒙版。

## 蒙版视口尺寸

该选项控制 Cubism Drawable 蒙版纹理尺寸，与上面的画布蒙版不同。界面提供“自动、256、512、1024、2048、4096”预设；
通常保持“自动”，仅在 Drawable 蒙版边缘确实模糊时提高数值，因为更高尺寸会增加显存占用。

## 清晰度与性能

正常混合、无滤镜、无画布蒙版时，模型绕过合成画布并保持原始清晰度。启用任何合成效果后，运行时根据模型显示尺寸动态计算合成分辨率，
单边上限为 8192，避免固定低分辨率合成造成的整体模糊。
