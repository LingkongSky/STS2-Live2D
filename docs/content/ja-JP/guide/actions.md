# アクションとホットキー

インポート時に model3 で宣言された Motion と Expression を収集します。VTube Studio モデルでは `.vtube.json`、`expressions`、
`animations` ディレクトリからアクションを補完し、統合した一覧をモデル詳細の「ホットキー」タブに表示します。

## Motion と Expression

- Motion は Idle、TapBody、Wave など時間軸を持つアニメーションです。
- Expression は smile、angry などの表情状態です。
- 自動 Idle は `Idle` という Motion Group を探します。
- Physics と Pose は個別に有効化できます。

一覧が空の場合は、model3 の宣言と参照ファイルの存在を確認してください。

## ホットキーを設定

1. 「モデル管理」を開きます。
2. 対象モデルの「詳細設定」を選びます。
3. 「ホットキー」タブを開きます。
4. キー、対象シーン、ループの有無を選びます。
5. 変更を保存します。

単独の文字、数字 `0–9`、ファンクションキー、キーの組み合わせを登録できます。

同じキーを複数アクションへ割り当てると、同時に実行される場合があります。UI には競合警告が表示されます。

## 全体表示ホットキー

全体表示ホットキーは一時的な表示コントロールです。1 回目で有効な全モデルを隠し、次の入力で各モデルのシーン表示状態を復元します。
モデル設定とモデルごとの有効状態は保持されます。

## 再生設定

- 負の再生速度は 0 として扱われます。
- アクションクールダウンは短時間の重複入力を防ぎます。
- ループ Motion は停止または置換されるまで続きます。
- Motion、Expression、Physics、Pose は後続フレームで Parameter を変更する場合があります。
